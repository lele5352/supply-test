python版本：3.X

依赖安装：
pip install -r requirements.txt



模块说明：

    -supply 根目录
    
        -barcodes 条形码生成文件夹
        
        -config 配置文件
        
        -data_helper 数据生成模块
        
        -logs 日志文件夹
        
        -script 脚本文件夹
        
        -testcase 测试用例文件夹，存放接口测试用例
        
        -utils 通用工具模块
